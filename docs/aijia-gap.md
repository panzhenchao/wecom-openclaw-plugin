# 企业微信 IM 功能差距（AIjia vs OpenClaw）

> 对比基准：AIjia lotus-app 当前实现 vs OpenClaw 官方企业微信插件

## AIjia 已有的功能

- 收发文字消息（私聊 + 群聊）
- Webhook 方式接收消息
- 扫码注册机器人
- 发送图片、文件

## AIjia 还没有的功能

| 功能 | 说明 |
|------|------|
| 流式回复 | AI 回复等全部生成完才发。OpenClaw 支持边生成边显示 |
| AI 卡片 | 回复只能是纯文字，没有富文本卡片展示 |
| 多账号 | 只能绑一个企业微信机器人 |
| 消息表情回应 | 无法用表情回应消息状态 |
| 媒体消息发送 | 只能收媒体，AI 回复里无法主动发图片/文件 |
